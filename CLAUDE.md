# AI-Ferðadagbók (AI Travel Journal)

## What this project is
A personal travel journal for a trip through Indonesia (May 1 – June 6).
Route: Jakarta → Tanjung Puting (Borneo) → Sulawesi → Flores → Jakarta.

Each day (or whenever I feel like it) I give Claude rough notes and Claude 
turns them into a polished, vivid journal entry saved as a Markdown file.

## Rules for Claude
- Write in vivid, warm, personal travel-writing style
- Never invent facts — only use what I provide in my notes
- If my notes are vague, ask rather than guess
- Keep it flexible — the structure will evolve as the trip goes on
- Save entries in an `entries/` folder, named by date + a short descriptive slug based on the content e.g `2025-05-03-orangutans-on-the-river.md`
